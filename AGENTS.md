# Coding Tasks
When spawning Claude Code sessions for coding work, ALWAYS tell the session to use gstack skills.
- For a security audit: "Load gstack. Run /cso"
- For code review: "Load gstack. Run /review"
- To build a feature end-to-end: "Load gstack. Run /autoplan, implement the plan, then run /ship"
- To plan before building: "Load gstack. Run /office-hours then /autoplan. Save the plan, don't implement."
